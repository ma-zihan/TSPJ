\section{Introduction}
The Traveling Salesman Problem with Job-times (TSPJ) is a hybrid problem that inherits properties from both the Traveling Salesman Problem(TSP) and the Scheduling Problem. TSPJ is a route optimization problem where a traveler must visit all nodes and each node exactly once and then return to the origin. Also, it inherits the scheduling properties when minimizing the completion time of the jobs.

In the paper \cite{MOSAYEBI2021105226}, the authors give a mathematical model for TSPJ. Efficient heuristics for the solution are required for solving large problems, and four heuristic procedures are proposed, which are based on the TSPJ-nearest neighbor(NN) heuristic, reverse assignment, TSPJ-2-opt, and local search improvement(LSI). These heuristics employ local search improvement techniques, which were developed specifically for the TSPJ, to enhance the solutions initially constructed by heuristics similar to the nearest neighbor method used in solving TSP.

The data and code in GAMS language has been provided in \url{https://github.com/TSPJLIB/TSPJ}.
